# Cronitor

!!! info
    This integration allows for notifications from [Cronitor](https://cronitor.io/) when issues are detected with your monitored jobs and services.

---

## Trigger Options

### Triggers

- `Issues` - Receive a notification when Cronitor detects an issue with a monitored job or service

### Channels

- Cronitor channel picker for each trigger

---

## Configuration

Click the **cog icon** to open the configuration options for Cronitor.

1. Notification pings for issue alerts
2. Notification layout customization

---

## Instructions

1. In Cronitor, navigate to your alert settings
2. Add a webhook notification pointing to your Notifiarr webhook URL
3. Enable the Issues trigger on the Notifiarr website
4. Test the notification to ensure it is working
