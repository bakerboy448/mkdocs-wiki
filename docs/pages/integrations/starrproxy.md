# Starr Proxy

!!! info
    This integration provides notifications when requests to your Starr applications (Sonarr, Radarr, etc.) are blocked by the Notifiarr proxy. Starr Proxy acts as a security layer that can filter and block unauthorized API requests.

---

## Trigger Options

### Triggers

- `Blocked` - Receive a notification when a request is blocked by the Starr Proxy

### Channels

- Starr Proxy channel picker for each trigger

---

## Configuration

Click the **cog icon** to open the configuration options for Starr Proxy.

1. Notification color for blocked requests
2. Notification pings for blocked requests

---

## Instructions

1. Enable the Starr Proxy integration on the Notifiarr website
2. Configure the Blocked trigger and select a notification channel
3. Set up your notification color and pings as desired
4. Test the notification to ensure it is working
