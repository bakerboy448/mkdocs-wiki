# Metrics

The Metrics page shows notification leaderboards across all Notifiarr users.

![metrics.png](../../assets/screenshots/website/metrics.png)

## Leaderboards

1. **Total** - Top 5 users by all-time total notifications sent
1. **Daily** - Top 5 users by notifications sent today
1. **Per Integration** - Top 5 users for each integration, sorted alphabetically. Each integration card also shows:
    - The number of active users for that integration (top right)
    - Today's notification count for that integration (footer)

Your own name is highlighted in red if you appear on any leaderboard.
